# 📒 Aplikasi Akuntansi

Aplikasi pencatatan keuangan sederhana berbasis **Flutter** untuk membantu mengelola pemasukan, pengeluaran, dan laporan keuangan bisnis kecil maupun personal.

---

## ✨ Fitur

### 🔐 Autentikasi
- Login dengan username/email dan password
- Sesi tersimpan otomatis (tetap login meski app ditutup)
- Logout dengan konfirmasi

### 📊 Dashboard *(coming soon)*
- Total pendapatan bulan ini
- Total pengeluaran bulan ini
- Selisih laba/rugi kas

### 💸 Transaksi *(coming soon)*
- Tambah pendapatan & pengeluaran
- Field: tanggal, kategori, deskripsi, nominal, metode bayar, catatan

### 📄 Laporan *(coming soon)*
- Filter rentang tanggal
- Export PDF & Excel

### ⚙️ Master Data *(coming soon)*
- Kategori pemasukan & pengeluaran
- Manajemen user

---

## 🚀 Memulai

### Prasyarat
- [Flutter SDK](https://flutter.dev/docs/get-started/install) >= 3.11.1
- Dart >= 3.11.1
- Android Studio / VS Code

### Instalasi

```bash
# Clone repository
git clone https://github.com/Lescy48/aplikasi-akuntansi.git
cd aplikasi-akuntansi

# Install dependencies
flutter pub get

# Jalankan aplikasi
flutter run
```
